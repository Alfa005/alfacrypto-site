# alfacrypto.ai

Crypto education website — explaining crypto simply, three times a week.

## Deploy

This site is hosted on [Netlify](https://netlify.com). Any push to the `main` branch auto-deploys.

## Structure

```
index.html    — Full single-page application (EN/FR bilingual)
netlify.toml  — Netlify configuration
_redirects    — SPA routing redirects
```
